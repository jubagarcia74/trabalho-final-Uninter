# Ciência de Dados ao Alcance dos Jovens

Plataforma web educacional sobre Ciência de Dados voltada para jovens do ensino fundamental e médio de escolas públicas de Porto Alegre/RS.

## Sobre o Projeto

Este projeto foi desenvolvido como parte da Atividade Extensionista II (Tecnologia Aplicada à Inclusão Digital - Projeto) do curso de CST em Ciência de Dados do Centro Universitário Internacional UNINTER.

O objetivo é democratizar o acesso ao conhecimento em Ciência de Dados, oferecendo conteúdo educacional gratuito e interativo para estudantes de escolas públicas.

## Seções do Site

1. **Aulas e Tutoriais** — Conceitos fundamentais de Ciência de Dados explicados de forma simples
2. **Exercícios Práticos** — Atividades interativas usando dados reais de Porto Alegre
3. **Introdução a Python** — Primeiros passos na linguagem mais usada em Ciência de Dados
4. **Dashboard Interativo** — Visualizações de dados educacionais reais (IBGE/INEP 2023)

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (ES6+)
- Chart.js (gráficos interativos)
- Google Fonts (Inter, JetBrains Mono)

## Dados Utilizados

Os dados apresentados no dashboard são reais e foram obtidos de fontes oficiais:
- IBGE Cidades (cidades.ibge.gov.br)
- INEP — Censo Escolar 2023
- IDEB — Índice de Desenvolvimento da Educação Básica
